# 오디오 파일 위치

발음 분석 페이지(`/pronunciation`)의 파형·음정·스펙트로그램 그래프가 분석할 오디오입니다.

## 기본 샘플

- 현재 기본값: **`sample.mp3`** (`/audio/sample.mp3` 로 서빙) — 페이지 첫 로드 시 이 파일이 분석됩니다.
- 파일명을 바꾸거나 `.wav`로 쓰려면 `src/pages/pronunciation/graphs/audio.ts`의 `DEFAULT_AUDIO_URL`만 수정하면 됩니다.

## 방문자 업로드

- 페이지의 **[파일]** 버튼으로 누구나 자기 음성을 올려 즉석에서 분석할 수 있습니다.
- 업로드 파일은 브라우저 메모리에서만 처리되고 서버로 전송되지 않습니다.

> 분석 정확도를 위해 짧은(2~8초) 발화 녹음을 권장합니다.
